Facial-expression-recognition-and-conversation-into-Emojis

  • Tech Stack: Python, Machine Learning, Computer Vision, TensorFlow, GUI.
  • Github URL: Project Link

Created an innovative project to recognize facial expressions and convert them into emojis. This system makes interacting with machines more expressive by automatically detecting emotions through facial expressions.

Implemented using Python, machine learning techniques, and a GUI interface with Tkinter, the project recognizes 7 major facial expressions. Key components include emotion_model.h5 for classification, haarcascade_frontalface_default.xml for face detection, and train.py for model training.

The project has high accuracy and smooth functionality. The code is available on GitHub for free download.
